# Download NEET PG 2025 Papers

Since NEET PG 2025 papers require manual download from websites, follow these instructions:

## Option 1: DigiNerve (Recommended)

**Best for:** Complete 200 recall questions with answers

1. Visit: https://www.diginerve.com/blogs/neet-pg-2025-recall-questions-with-answers-free-pdf-download-all-200-qs/
2. Scroll down to find the **"PDF Download link"** button
3. Click to download from Google Drive
4. Save the PDF to `solved-papers/` folder as `NEET-PG-2025-Recall-Questions-With-Answers.pdf`

**Contents:**
- All 200 recall questions
- Subject-wise organization
- Detailed answers

## Option 2: Shiksha.com

**Best for:** Official answer key reference

1. Visit: https://www.shiksha.com/medicine-health-sciences/articles/neet-pg-2025-question-paper-with-answer-key-solutions-pdf-download-blogId-207156
2. Look for download buttons/links on the page
3. Download the PDF
4. Save to `solved-papers/` folder

## Option 3: Official NBE Answer Key

**Best for:** Official verification

1. Visit: https://www.shiksha.com/medicine-health-sciences/neet-pg-exam-answer-key
2. Download the official answer key (released August 29, 2025)
3. Save to `question-papers/` folder

## After Download

Once you've downloaded the files:

1. **Rename files** to match naming convention:
   - `NEET-PG-2025-Question-Paper.pdf`
   - `NEET-PG-2025-Solutions.pdf`
   or
   - `NEET-PG-2025-Question-Paper-With-Solutions.pdf`

2. **Place in correct folder:**
   - Questions only → `question-papers/`
   - With solutions → `solved-papers/`

3. **Verify download:**
   ```bash
   ls -lh solved-papers/ | grep 2025
   ```

## Important Notes

- NEET PG 2025 was conducted on **August 3, 2025**
- Papers available are **memory-based** (reconstructed by students/experts)
- Official NBE answer key was released on **August 29, 2025**
- Some websites may require free registration to download

## Troubleshooting

**If download links don't work:**
1. Try accessing from a different browser
2. Disable ad-blockers temporarily
3. Check if the website requires registration
4. Try alternative sources listed above

**File size reference:**
- Question papers: ~300KB - 1.5MB
- Solved papers: ~500KB - 2MB
